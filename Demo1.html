Introduction to HTML

HTML, which stands for HyperText Markup Language, was first introduced in 1991 by Tim Berners-Lee, a scientist at CERN, to help researchers share documents over the internet. The first version of HTML was very basic, containing only a small number of tags for formatting text and adding simple structure to web documents. In 1995, HTML 2.0 was released and standardized by the Internet Engineering Task Force (IETF).

History of HTML

By 1997, HTML was taken over by the World Wide Web Consortium (W3C), which released HTML 3.2. This version added features like tables, scripting (JavaScript), and support for CSS (Cascading Style Sheets), making websites more dynamic and styled. Two years later, in 1999, HTML 4.01 was introduced with a focus on separating content from design. It offered different document types such as Strict, Transitional, and Frameset, and laid the foundation for accessibility and semantic structure.

In 2008,   work on HTML5 began as a collaboration between W3C and WHATWG (Web Hypertext Application Technology Working Group). HTML5 brought a major transformation, introducing modern features such as video, audio, canvas, and semantic tags like article and section. It improved support for mobile devices, multimedia, and interactive applications. Today, HTML5 is the backbone of modern web development, enabling developers to build rich, responsive, and accessible web applications.

Different tags in HTML

1. Heading Tags (h1 to h6)

  • Used to define headings or titles on a webpage.
  • h1 is the largest, h6 is the smallest.
  • Helps organize content and improves SEO (Search Engine Optimization).

2. Paragraph Tag

  • Used to write a block of text (a paragraph).
  • Browsers add space above and below the paragraph automatically.

3. Break Line Tag

  • Adds a line break (moves text to the next line).
  • It’s an empty tag (no closing tag needed).

4. Preformatted Text Tag

  • Displays text exactly as it is written (preserves spaces, tabs, and line breaks).
  • Useful for showing code, poems, or ASCII art.
 
    Jonny jonny yes papa...
       Eating sugar no papa...
         Telling lies no papa...
           Open your mouth HA HA HA....

5. Horizontal Rule Tag

  • Inserts a horizontal line across the page.
  • Used to separate sections of content.

6. Bold Tag

  • Makes the text bold.
  • Used for styling, no extra importance given to the text.

    • eg:

    <b>This is bold</b>

    This is bold

7.Strong Importance

  • Makes the text bold and conveys importance or emphasis.
  • Carries sematic meaning

    • eg :

    <strong>This is important text</strong>

    This is important text

8.Italic

  • Makes the text italic.
  • Used for styling, like technical terms or names.

    • eg :

    <i>This is italic text</i>

    This is italic text

9.Emphasis

  • Emphasizes the text and makes it italic.
  • Has semantic meaning (stress or importance).

    • eg :

    <em>This is emphasized text</em>

    This is emphasized text

10.Underline

  • Underlines the text.
  • Used rarely, often avoided in modern web design.

    • eg :

    <em>This is underlined text</em>

    This is underlined text

11.Quotation Block

  • Displays a block-level quotation (usually indented).

    • eg :

    <blockquote>This is a long quoted section.</blockquote>

    This is a long quoted section.

12.Inline Quotation

  • Displays short quotes inside double quotation marks.

    • eg :

    <q>This is a short quote.</q>

    This is a short quote.

13.Strike Through (Deleted Text)

  • Shows text that has been deleted or no longer valid.

    • eg :

    <del>This text is deleted.</del>

    This text is deleted.

14.Strike Through (No Longer Accurate)

  • Used for inaccurate or outdated text.

    • eg :

    <strike>This text is deleted.</strike>

    This is incorrect text.

15.Small Text

  • Makes the text appear smaller than normal.

    • eg :

    <smalle>This is small text.</small>

    This is small text.

16.Big Text (Deprecated)

  • Makes the text appear larger than normal.
  • This tag is deprecated in HTML5. Use CSS instead.

    • eg :

    <big>This is big text.</big>

    This is big text.

17.Subscript

  • Displays text below the normal line (subscript).

    • eg :

    <p>H<sub>2</sub>O</p>

    H2O

18.Superscript

  • Displays text above the normal line (superscript).

    • eg :

    <p>H<sup>2</sup></p>

    H2

Attributes

    Attributes provide additional information about HTML elements. They are always specified in the start tag and come in name/value pairs like name="value".

    Synatx :

    <p id="intro" class="highlight" title="About Us"</p>

    Welcome to our website!

19.Images Tag

    The image tag is used to embed images in a webpage. It is an empty tag (doesn't have a closing tag).

    eg :

    <img src="image.jpg" alt="Descriptiont" width="300" height="200">

Important Attributes:
  • src- Path to the image (required).
  • alt- Alternate text for accessibility
  • width- Width of the image (in px or %)
  • height- Height of the image
  • title- Tooltip on hover
  • loading- Lazy loading (lazy, eager, auto)
  • style- Inline styling (optional)

20 HTML (Anchor) Tag

    The anchor tag defines a hyperlink, used to navigate to another page, file, or section.

    eg :

    <a href="resume.pdf" download>Download Resume</a>
<a href="https://google.com" target="_blank" title="Go to Google">Google</a>
    Download Resume Google
Important Attributes:
  • _self - Opens in same tab (default)
  • _blank - Opens in new tab
  • _parent - Opens in parent frame
  • _top - Opens in full window

HTML Lists

HTML supports three types of lists:

  1. Ordered List
  2. Unordered List
  3. Description List

21.Ordered List

Used when the order of items matters (like steps, rankings).

Eg :

<ol type="A" start="3">
  <li>Item One</li>
  <li>Item Two</li>
</ol>
  1. Item One
  2. Item Two
type Attribute (Representation):
  • 1 - 1, 2, 3, ... (default)
  • A - A, B, C, ...
  • a - a, b, c, ...
  • I - I, II, III, ...
  • i - i, ii, iii, ...
